adwaita-icon-theme-44.0

Introduction to Adwaita Icon Theme

The Adwaita Icon Theme package contains an icon theme for GTK+ 3 and GTK 4applications.

Note

Development versions of BLFS may not build or run some packages properly if dependencies have been updated since the most recent stable versions of the book.

Package Information

  • Download (HTTP): https://download.gnome.org/sources/adwaita-icon-theme/44/adwaita-icon-theme-44.0.tar.xz

  • Download (FTP): ftp://ftp.acc.umu.se/pub/gnome/sources/adwaita-icon-theme/44/adwaita-icon-theme-44.0.tar.xz

  • Download MD5 sum: 1abae78e6cd50d4055b5c88ffe8b56a5

  • Download size: 2.4 MB

  • Estimated disk space required: 31 MB

  • Estimated build time: 0.1 SBU (Using parallelism=4)

Adwaita Icon Theme Dependencies

Optional

git-2.40.1, GTK+-2.24.33 or GTK+-3.24.37 (if present, librsvg-2.56.0 is also required, and gtk-update-icon-cache and gtk-encode-symbolic-svg are run after installing), Inkscape-1.2.2 and Icon Tools

User Notes: https://wiki.linuxfromscratch.org/blfs/wiki/adwaita-icon-theme

Installation of Adwaita Icon Theme

Install Adwaita Icon Theme by running the following commands: 

./configure --prefix=/usr &&
make

This package does not come with a test suite.

Now, as the root user: 

make install

Contents

Installed Programs: None
Installed Library: None
Installed Directory: /usr/share/icons/Adwaita
