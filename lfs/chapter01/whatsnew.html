1.2. What's new since the last release

In the 11.3 release, --enable-default-pie and --enable-default-ssp are enabled for GCC. These techniques can mitigate some malicious attacks, but they do not provide perfect security. Note that some textbooks assume these options are disabled, so that if you run examples from such a textbook on an LFS system, you may need to disable PIE and SSP with the GCC options -fno-pie -no-pie -fno-stack-protection.
