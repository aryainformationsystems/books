1.3. Changelog

This is version r11.3-65 of the Linux From Scratch book, dated April 16th, 2023. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via https://www.linuxfromscratch.org/mirrors.html.
